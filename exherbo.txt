3. What makes it "Plus"?

When you start writing this exheres, you'll notice a few unique variables you might want to toggle:

    GOP_FIX: Some versions allow you to bake in a "GOP Fix" that forces a specific resolution at compile time. For your Zenbook Duo's 1080p screen, this ensures the buffer is exactly 1920x1080.

    DMA_PROTECT: A security feature (though you prefer performance) that handles how memory is accessed during the boot phase.

    BDS_PROTOCOL: This is the "Boot Device Selection" logic. rEFIndPlus's version is much stricter, which is what gives you that "Strict UEFI Compliance."
